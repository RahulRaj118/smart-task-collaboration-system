# Smart Task and Collaboration System

A Spring Boot backend application designed to manage tasks and enable team collaboration.

## Features
- User management
- Task creation and assignment
- Task status tracking
- RESTful APIs
- Global exception handling
- Layered architecture (Controller, Service, Repository)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Project Structure
- controller – REST endpoints
- service – business logic
- repository – data access layer
- dto – request/response models
- exception – global error handling
- config – application configuration

## How to Run
### Prerequisites
- Java 21 installed
- MySQL installed and running
- Maven (optional, Maven Wrapper is included)

### Steps to Run the Project
1. Clone the repository:
   git clone https://github.com/RahulRaj118/smart-task-collaboration-system.git
